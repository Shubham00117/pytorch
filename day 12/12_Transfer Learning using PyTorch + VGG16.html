02 · What is Transfer Learning
📖
Definition

A ML technique where a model trained on one task is reused (partially or fully) for a different but related task. Instead of training from scratch (expensive, needs large data), TL leverages knowledge from a pre-trained model to improve efficiency.

Why DL needs TL
  • Data hungry — needs millions of samples
  • Costly — requires GPU / graphics card
How it works
  • Pretrain on large dataset (ImageNet, 1.4M imgs)
  • Fine-tune on smaller, specific dataset
  • Some layers frozen, others retrained

06 · Data Pipeline — Fashion-MNIST → VGG Input
🔄
8-Step Transform: (1,784) → (3,224,224)

Fashion-MNIST is tabular (1×784). VGG16 expects (3, 224, 224). We must convert!

1reshape(28,28) → 2D matrix → PIL(28,28)
2astype(np.uint8) — change datatypeuint8
3np.stack([image]*3, axis=-1) — grayscale → color(28,28,3)
4Image.fromarray(image) — array → PILPIL (H,W,C)
5transforms.Resize(256)(3,256,256)
6transforms.CenterCrop(224)(3,224,224)
7transforms.ToTensor() — scale to [0,1](0,1)
8Normalize(mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225])p=(p−μ)/σ
⚠️ np.stack trick: np.stack([arr]*3, axis=-1) → shape (H,W,3) | np.stack([arr]*3) → shape (3,H,W)
07 · Complete Implementation Code
💻
Transforms + CustomDataset
Transforms (ImageNet normalization)
from torchvision.transforms import transforms

custom_transform = transforms.Compose([
    transforms.Resize(256),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406],
                         std=[0.229, 0.224, 0.225])
])
CustomDataset — handles tabular → image conversion
class CustomDataset(Dataset):
    def __init__(self, features, labels, transform):
        self.features  = features
        self.labels    = labels
        self.transform = transform

    def __len__(self):
        return len(self.features)

    def __getitem__(self, index):
        # 1) reshape to 2D
        image = self.features[index].reshape(28, 28)

        # 2) change datatype
        image = image.astype(np.uint8)

        # 3) B&W → color (H,W,C) format
        image = np.stack([image]*3, axis=-1)

        # 4) convert array → PIL image
        image = Image.fromarray(image)

        # 5) apply transforms (resize, crop, tensor, normalize)
        image = self.transform(image)

        return image, torch.tensor(self.labels[index], dtype=torch.long)

# Instantiate
train_dataset = CustomDataset(X_train, y_train, transform=custom_transform)
test_dataset  = CustomDataset(X_test,  y_test,  transform=custom_transform)

train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True,  pin_memory=True)
test_loader  = DataLoader(test_dataset,  batch_size=32, shuffle=False, pin_memory=True)

VGG16 Setup — Load, Freeze, Replace Classifier 
from torchvision import models

# Step 1: Load pretrained VGG16 (downloads 528MB)
vgg16 = models.vgg16(pretrained=True)

# Step 4: Freeze all feature extraction layers
for param in vgg16.features.parameters():
    param.requires_grad = False

# Step 3: Replace classifier with custom 10-class head
vgg16.classifier = nn.Sequential(
    nn.Linear(25088, 1024),
    nn.ReLU(),
    nn.Dropout(0.5),
    nn.Linear(1024, 512),
    nn.ReLU(),
    nn.Dropout(0.5),
    nn.Linear(512, 10)    # 10 Fashion-MNIST classes
)

# Move to GPU
vgg16 = vgg16.to(device)
🚀
Training & Evaluation 
# Hyperparameters
learning_rate = 0.0001   # small LR for fine-tuning!
epochs        = 10

criterion = nn.CrossEntropyLoss()
# ⚠️ Optimize ONLY classifier params (feature layers are frozen)
optimizer = optim.Adam(vgg16.classifier.parameters(), lr=learning_rate)

# Training loop
for epoch in range(epochs):
    total_epoch_loss = 0
    for batch_features, batch_labels in train_loader:
        batch_features, batch_labels = \
            batch_features.to(device), batch_labels.to(device)

        outputs = vgg16(batch_features)          # forward pass
        loss    = criterion(outputs, batch_labels) # calc loss

        optimizer.zero_grad()                     # back pass
        loss.backward()
        optimizer.step()

        total_epoch_loss += loss.item()

    avg_loss = total_epoch_loss / len(train_loader)
    print(f'Epoch: {epoch+1}, Loss: {avg_loss}')

# Evaluation (test data)
vgg16.eval()
total = 0; correct = 0
with torch.no_grad():
    for batch_features, batch_labels in test_loader:
        batch_features, batch_labels = \
            batch_features.to(device), batch_labels.to(device)
        outputs = vgg16(batch_features)
        _, predicted = torch.max(outputs, 1)
        total   += batch_labels.shape[0]
        correct += (predicted == batch_labels).sum().item()

print(correct / total)   # accuracy
08 · Key Notes & Gotchas
⚠️
Critical Points
  • Use very small LR (0.0001 not 0.01) — classifier is fresh
  • Optimizer on vgg16.classifier.parameters() only — not all params
  • Fashion-MNIST is grayscale — must convert to 3-channel for VGG
  • np.stack([image]*3, axis=-1) → (H,W,C), not (C,H,W)
  • Use ImageNet mean/std for normalization — not 0.5/0.5
  • VGG16 input must be at least 32×32, expects 224×224
🔑
Quick Reference
  • vgg16.features — all conv/pool layers
  • vgg16.classifier — FC layers (replaceable)
  • param.requires_grad=False — freeze layer
  • 25088 = 512 × 7 × 7 (after AdaptiveAvgPool)
  • VGG trained on ImageNet — 1000 classes, 1.4M images
  • Epoch 1 loss ≈ 0.645 (convergence fast with TL)
