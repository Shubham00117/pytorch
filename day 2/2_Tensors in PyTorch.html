What are Tensors
▾

Tensors are multi-dimensional arrays — the fundamental data structure in PyTorch. Very similar to NumPy arrays but with GPU support.

Why Tensors are Useful & Where They're Used
▾
Why Useful
  • Math Operations — add, multiply, dot product for neural nets
  • Real-world Data — images, audio, video, text all represented as tensors
  • GPU Acceleration — optimized for CUDA/TPUs, parallel computation
In Deep Learning
  • Data Storage — training data stored as tensors
  • Weights & Biases — learnable parameters are tensors
  • Matrix Ops — matmul, dot product, broadcasting
  • Training — forward pass tensors flow through; gradients computed in backward pass
Activation function formula: activation(W·x + b) — W, x, b are all tensors.

Creating Tensors
▾
Setup
import torch
print(torch.__version__)  # 2.5.1+cu121

if torch.cuda.is_available():
    print(torch.cuda.get_device_name(0))  # Tesla T4
Construction Methods
# Uninitialized (garbage values)
a = torch.empty(2, 3)              # type: torch.Tensor

# Filled with constant values
torch.zeros(2, 3)                 # all 0.0
torch.ones(2, 3)                  # all 1.0
torch.full((3, 3), 5)             # all 5

# Random
torch.rand(2, 3)                  # uniform [0, 1)
torch.randn(2, 3)                 # standard normal
torch.randint(size=(2,3), low=0, high=10)

# Reproducibility
torch.manual_seed(100)
torch.rand(2, 3)   # same result every time with same seed

# From data
torch.tensor([[1,2,3], [4,5,6]])   # tensor([[1,2,3],[4,5,6]])

# Ranges & sequences
torch.arange(0, 10, 2)             # tensor([0,2,4,6,8])
torch.linspace(0, 10, 10)          # 10 evenly spaced vals
torch.eye(5)                        # 5×5 identity matrix

Tensor Shapes & Data Types
▾
Shapes
x = torch.tensor([[1,2,3],[4,5,6]])
x.shape                              # torch.Size([2, 3])

# Create new tensors with same shape as x
torch.empty_like(x)                  # uninitialized, same shape
torch.zeros_like(x)                  # all zeros
torch.ones_like(x)                   # all ones

# Check & assign dtype
x.dtype                              # torch.int64 (default for int)
torch.tensor([1.0,2.0,3.0], dtype=torch.int32)

# Convert dtype
x.to(torch.float32)

Scalar Operations (Broadcast)
▾

A scalar is applied to every element of the tensor.

x = torch.rand(2, 2)
# tensor([[0.9594, 0.1334], [0.8591, 0.6639]])

x + 2          # addition    → [[2.9594, 2.1334], ...]
x - 2          # subtraction
x * 3          # multiplication
x / 3          # division
(x * 100)//3   # integer division  → [[31., 4.], [28., 22.]]
((x*100)//3)%2 # modulo            → [[1., 0.], [0., 0.]]
x**2           # power             → [[0.9205, 0.0178], ...]

Element-wise & Unary Operations
▾
Element-wise (same shape tensors)
a = torch.rand(2, 3)
b = torch.rand(2, 3)

a + b   # element-wise add
a - b   # subtract
a * b   # multiply  (NOT matrix multiply!)
a / b   # divide
a ** b  # power
a % b   # modulo
Unary Operations
c = torch.tensor([1, -2, 3, -4])

torch.abs(c)   # tensor([1, 2, 3, 4])
torch.neg(c)   # tensor([-1, 2, -3, 4])
Rounding Operations
d = torch.tensor([1.9, 2.3, 3.7, 4.4])

torch.round(d)              # [2., 2., 4., 4.]
torch.ceil(d)               # [2., 3., 4., 5.]
torch.floor(d)              # [1., 2., 3., 4.]
torch.clamp(d, min=2, max=3) # [2.0, 2.3, 3.0, 3.0]
clamp clips values to [min, max] range — useful for gradient clipping.

Reduction Operations
▾
e = torch.randint(size=(2,3), low=0, high=10)
# e.g. tensor([[5,5,8],[8,5,0]])

torch.sum(e)              # total sum of all elements
torch.sum(e, dim=0)       # sum along columns (↓) → tensor([13, 10, 8])
torch.sum(e, dim=1)       # sum along rows    (→) → tensor([18, 13])

torch.mean(e.float())     # overall mean
torch.mean(e.float(), dim=0) # mean per column → [4., 4., 7.]

torch.median(e)           # tensor(5.)
torch.max(e)              # max value
torch.min(e)              # min value
torch.prod(e)             # product of all  → tensor(3780.)
torch.std(e.float())      # std deviation   → tensor(3.0332)
torch.var(e.float())      # variance        → tensor(9.2000)

torch.argmax(e)           # index of max (flattened) → tensor(2)
torch.argmin(e)           # index of min (flattened) → tensor(0)
dim=0 → operate across rows (↓ columns). dim=1 → operate across columns (→ rows).

Matrix Operations
▾
f = torch.randint(size=(2,3), low=0, high=10)
g = torch.randint(size=(3,2), low=0, high=10)

# Matrix multiplication  (2×3 @ 3×2 = 2×2)
torch.matmul(f, g)         # tensor([[63,73],[79,88]])

# Dot product (1D vectors)
v1 = torch.tensor([1,2])
v2 = torch.tensor([3,4])
torch.dot(v1, v2)          # 1×3 + 2×4 = 11

# Transpose  (swaps dim 0 and 1)
torch.transpose(f, 0, 1)   # (2×3) → (3×2)

# Determinant & Inverse  (needs square float matrix)
h = torch.randint(size=(3,3), low=0, high=10, dtype=torch.float32)

torch.det(h)              # tensor(224.)
torch.inverse(h)          # 3×3 inverse matrix

Comparison & Special Functions
▾
Comparison (element-wise → bool tensor)
i = torch.randint(size=(2,3), low=0, high=10)
j = torch.randint(size=(2,3), low=0, high=10)

i > j   # greater than
i < j   # less than
i == j  # equal to
i != j  # not equal to
i >= j  # greater than or equal
i <= j  # less than or equal
# Returns tensor([[False, True, ...], ...])
Special / Activation Functions
k = torch.randint(size=(2,3), low=1, high=10, dtype=torch.float32)

torch.log(k)              # natural log (ln)
torch.exp(k)              # e^x
torch.sqrt(k)             # square root

# Activation functions
torch.sigmoid(k)          # σ(x) = 1/(1+e^-x) → [0,1]
torch.softmax(k, dim=0)  # probabilities along dim (sum=1)
torch.relu(k)             # max(0, x)

Inplace Operations
▾
Rule: Methods ending with _ modify the tensor in-place — no new tensor is created, saving memory.
m = torch.rand(2, 3)
n = torch.rand(2, 3)

# Out-of-place (creates new tensor)
m + n                   # m is unchanged

# Inplace (modifies m directly)
m.add_(n)               # m = m + n
m.relu_()               # m = relu(m)
m.mul_(2)               # m = m * 2

Copying a Tensor
▾
⚠ Shallow Copy (shared memory)
a = torch.rand(2, 3)
b = a          # same data!

a[0][0] = 0
# b[0][0] is also 0 ← danger!
# id(a) ≠ id(b) but same storage
✓ Deep Copy (independent)
b = a.clone()  # new memory

a[0][0] = 10
# b is unchanged ✓
# Different storage locations
Always use .clone() when you want an independent copy that won't be affected by changes to the original.

Tensor Operations on GPU
▾
torch.cuda.is_available()         # True
device = torch.device('cuda')

# Create tensor directly on GPU
torch.rand((2,3), device=device)
# tensor([[...]], device='cuda:0')

# Move existing CPU tensor to GPU
a = torch.rand(2, 3)
a.to(device)
# tensor([[...]], device='cuda:0')
GPU vs CPU Benchmark (10000×10000 matmul)
import time

size = 10000
cpu1, cpu2 = torch.randn(size,size), torch.randn(size,size)

t = time.time()
torch.matmul(cpu1, cpu2)
cpu_time = time.time() - t        # ~17.7 seconds

gpu1 = cpu1.to('cuda'); gpu2 = cpu2.to('cuda')
t = time.time()
torch.matmul(gpu1, gpu2)
torch.cuda.synchronize()
gpu_time = time.time() - t        # ~0.57 seconds

print(cpu_time / gpu_time)         # Speedup: ~31×
GPU is ~31× faster for large matrix multiplications — critical for deep learning training.

Reshaping Tensors
▾
a = torch.ones(2, 2, 2)    # shape [2,2,2] — 8 elements

# reshape — change shape (total elements must match)
a.reshape(2,2,2,2)          # ← wait, needs matching size
a.reshape(4, 2)             # → shape [4, 2]

# flatten — collapses all dims to 1D
a.flatten()                 # tensor([1.,1.,1.,1.,1.,1.,1.,1.])

# permute — reorder dimensions
b = torch.rand(2, 3, 4)     # shape [2, 3, 4]
b.permute(2, 0, 1).shape    # torch.Size([4, 2, 3])
b.permute(2, 1, 0).shape    # torch.Size([4, 3, 2])

# unsqueeze — adds a dimension of size 1
c = torch.rand(226, 226, 3)  # single image [H,W,C]
c.unsqueeze(0).shape         # torch.Size([1,226,226,3]) — batch of 1

# squeeze — removes dimensions of size 1
d = torch.rand(1, 20)
d.squeeze(0).shape           # torch.Size([20])
Method What it does Example
reshape() Change shape, same data [8] → [2,4]
flatten() Collapse all dims → 1D [2,3,4] → [24]
permute() Reorder dimensions [2,3,4] → [4,2,3]
unsqueeze(dim) Insert dim of size 1 [226,226,3] → [1,226,226,3]
squeeze(dim) Remove dim of size 1 [1,20] → [20]

NumPy ↔ PyTorch
▾
import numpy as np

# Tensor → NumPy
a = torch.tensor([1, 2, 3])
b = a.numpy()               # array([1, 2, 3])
type(b)                      # numpy.ndarray

# NumPy → Tensor
c = np.array([1, 2, 3])
torch.from_numpy(c)          # tensor([1, 2, 3])
Note: On CPU, .numpy() shares memory with the tensor. Modifying one modifies the other. Use .clone().numpy() to avoid this.
