Steps for GPU Training
1

Check GPU Availability

Detect if CUDA is available. If yes, use cuda; otherwise fall back to cpu.

python 
import torch

# Check for GPU
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
print(f"Using device: {device}")
2

Move the Model to GPU

Move your model to the selected device (cuda or cpu) so all computations occur on the same device.

python 
model = MyNN(X_train.shape[1])  # instantiate model
model = model.to(device)          # move model to GPU

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(model.parameters(), lr=learning_rate)
3

Modify the Training Loop — Move Data to GPU

Each batch of features and labels must be moved to the GPU before processing. Both model and data must be on the same device.

python 
for epoch in range(epochs):
    total_epoch_loss = 0
    for batch_features, batch_labels in train_loader:
        # Move data to GPU
        batch_features, batch_labels = \
            batch_features.to(device), batch_labels.to(device)
        # Forward pass
        outputs = model(batch_features)
        loss = criterion(outputs, batch_labels)
        # Back pass
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
4

Modify the Evaluation Loop — Move Data to GPU

Same pattern for test data. Use torch.no_grad() to disable gradient computation for efficiency.

python 
total = 0; correct = 0

with torch.no_grad():
    for batch_features, batch_labels in test_loader:
        # Move data to GPU
        batch_features, batch_labels = \
            batch_features.to(device), batch_labels.to(device)
        outputs = model(batch_features)
        _, predicted = torch.max(outputs, 1)
        total = total + batch_labels.shape[0]
        correct = correct + (predicted == batch_labels).sum().item()

print(correct / total)
Step 5 — Optimize GPU Usage
a. Use Larger Batch Sizes

Larger batches better utilise GPU memory and reduce computation time per epoch.

python 
train_loader = DataLoader(
    train_dataset,
    batch_size=128,
    shuffle=True,
    pin_memory=True
)
test_loader = DataLoader(
    test_dataset,
    batch_size=128,
    shuffle=False,
    pin_memory=True
)
b. Enable DataLoader Pinning

Use pin_memory=True to speed up CPU → GPU data transfer via pinned (page-locked) memory.

python 
train_loader = DataLoader(
    train_dataset,
    batch_size=32,
    shuffle=True,
    pin_memory=True  # ← key
)
