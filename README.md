# 🔥 PyTorch Mastery: A 14-Day Deep Learning Journey

Welcome to the **PyTorch Mastery** repository! This project documents a comprehensive 14-day journey into deep learning using PyTorch, covering everything from fundamental tensor operations to advanced architectures like CNNs, RNNs, and LSTMs.

---

## 📅 Roadmap & Course Content

| Day | Topic | Key Concepts |
| :--- | :--- | :--- |
| **Day 1** | [Introduction to PyTorch](day%201) | Getting started, Installation, and Core philosophy. |
| **Day 2** | [Tensors in PyTorch](day%202) | Creation, operations, and manipulation of Tensors. |
| **Day 3** | [PyTorch Autograd](day%203) | Automatic differentiation and gradient computation. |
| **Day 4** | [Training Pipeline](day%204) | Full workflow: Data -> Model -> Loss -> Optimizer. |
| **Day 5** | [PyTorch `nn` Module](day%205) | Building layers, activation functions, and modules. |
| **Day 6** | [Dataset & DataLoader](day%206) | Efficient data loading and custom dataset handling. |
| **Day 7** | [Building an ANN](day%207) | Designing and training a Multi-Layer Perceptron (MLP). |
| **Day 8** | [Training on GPU](day%208) | Accelerating computations using CUDA and `to(device)`. |
| **Day 9** | [Optimizing the Network](day%209) | Learning rates, schedulers, and optimization algorithms. |
| **Day 10** | [Hyperparameter Tuning](day%2010) | Automated search using **Optuna**. |
| **Day 11** | [Building a CNN](day%2011) | Convolutional layers, pooling, and image classification. |
| **Day 12** | [Transfer Learning](day%2012) | Leveraging **VGG16** for complex vision tasks. |
| **Day 13** | [RNN-Based QA System](day%2013) | Recurrent Neural Networks for Text Analysis. |
| **Day 14** | [LSTM Word Predictor](day%2014) | Long Short-Term Memory for sequential prediction. |

## 🚀 Getting Started

### Prerequisites

To run these notebooks (HTML exports included here), ensure you have the following installed:
- Python 3.8+
- PyTorch
- NumPy
- Optuna (for Day 10)
- Matplotlib (for visualization)

### Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Shubham00117/pytorch.git
   ```
2. Navigate to the specific day folder to view the content:
   ```bash
   cd "PyTorch/day 1"
   ```
